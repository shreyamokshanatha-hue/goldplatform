# Gold Tracking Project

## Overview
This project is designed to **track gold prices, purchases, and sales**. It allows users to monitor gold market trends, maintain personal records of gold investments, and generate summaries of their gold holdings.

---

## Features
- Track daily gold price fluctuations.
- Record gold purchases and sales.
- Calculate total investment and current value.
- Generate reports and summaries.
- User-friendly interface.
